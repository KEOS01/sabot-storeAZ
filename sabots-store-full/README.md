# Sabots Store

Démo e‑commerce (Next.js + Tailwind + shadcn/ui).

## Démarrage en local

1. Installer les dépendances :
   ```bash
   npm install
   ```

2. Lancer le serveur de dev :
   ```bash
   npm run dev
   ```
   → [http://localhost:3000](http://localhost:3000)

## Déploiement

- Pousser ce repo sur **GitHub**
- Connecter à **Vercel**
- Cliquer **Deploy**
